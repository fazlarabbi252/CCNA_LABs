## 01 Router Static Routing , ARP , TCP , HTTP Simulation

This project demonstrates static routing and Address Resolution Protocol (ARP) behavior using Cisco Packet Tracer.

## Network Configuration Details
- **Laptop0:** 192.168.1.100 (Gateway: 192.168.1.1)
- **Server0:** 192.168.2.200 (Gateway: 192.168.2.1)
- **Router0 & 1:** 10.10.10.1 & 10.10.10.2
- **Router Link:** 10.10.10.0/24

## How to Test
1. Open `Static_Routing.pkt` in Cisco Packet Tracer.
2. Open Laptop0's Command Prompt and run: `ping 192.168.2.200`
