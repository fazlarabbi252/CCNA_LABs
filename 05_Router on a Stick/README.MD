## 05 cisco Router on a stick configurtion

This project demonstrates Router on a stick.

## Network Configuration Details
- **Laptop0 - 5:** use as a user devices
- **Switch0 - 2:** use for VLANs & Trunk port.
- **Router0:** use as `Router on a stick` or `Inter VLANs Routing`


## How to Test
1. Open `Router on a stick.pkt` in Cisco Packet Tracer.
2. First trun on `simulation` mode.
3. Open a laptop and `ping 192.168.20.10` or select a `Add Simple PDUs` .
4. Then analysis the packet that is going successfully or not.
