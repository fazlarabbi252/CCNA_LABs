## 04 cisco SSH configurations

This project demonstrates how to use `SSH to access Router` in a network.

## Network Configuration Details
- **Laptop0:** use for configure the SSH
- **Router0:** use as a SSH access device

## How to Test
1. Open `SSH Configure.pkt` in Cisco Packet Tracer.
2. password of SSH is :`123`
3. In client laptop command prompt run : `ssh -l admin 192.168.10.1` to access the Router with SSH or use the `Terminal`
4. Now put the password. And you are able to access the Router
