## 🚀 How to Use This Repository

1. **Clone or Download:**  
   Clone this repository or download the specific `.pkt` file from the lab folders.

2. **Open the Project:**  
   Open the file using **Cisco Packet Tracer** (Latest Version Recommended).

3. **Verify and Run:**  
   Inside the lab folder, follow the step-by-step verification guide in the respective `README.md`.
